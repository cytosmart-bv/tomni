Annotation Manager
=========================

The Annotation Manager is a versatile tool for managing and processing annotations in diverse formats. 
Annotations can be provided in various representations, including contours, JSON objects, masks, and more. 
This class facilitates the extraction of features from these annotations and offers the flexibility to convert between different annotation formats.
