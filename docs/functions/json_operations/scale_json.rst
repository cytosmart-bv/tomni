scale_json
=================
.. autofunction:: your_module_name.scale_json
   :members:
   :undoc-members:

This function scales a JSON object (ellipse or polygon) by a specified scaling factor.

**Parameters:**

- ``json_object`` (dict): A JSON object following the standard format.
- ``scaling_factor`` (float): The multiplier to scale the object.

**Returns:**

- ``dict``: A new JSON object that is scaled by the specified factor.

**Raises:**

- ``ValueError``: If the object type is not supported.
