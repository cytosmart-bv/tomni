add_area
=================

.. autofunction:: your_module_name.add_area
   :members:
   :undoc-members:

This function calculates the area of an object based on its type (circle, ellipse, or polygon) and adds the computed area to the input JSON object.

**Parameters:**

- ``json_object`` (dict): The JSON object following the standard CytoSMART format.

**Raises:**

- ``ValueError``: If the object type is not supported.
