Tomni
=====

Tomni is created by CytoSMART

# Features

* Bounding box fitting (bbox fitting)
* Illumination correction 
  * Brightfield
  * Fluorescence
* Json operation
  * Add circularity property
  * Scale object
  * Yranslate object
* Make a mask
  * Ellipse
  * Polygon
* Shape fitting
  * Rect around ellipse
* Transformers of data format
  * Contours to json
  * Ellipse to json
  * Json 2 contours
  * Json 2 label
  * Labels 2 contours
  * Labels 2 lists of points
